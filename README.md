# Dashboard Business Intelligence - Butterfly Effect

## Deskripsi Proyek

Dashboard ini dikembangkan menggunakan Power BI untuk menganalisis kinerja penjualan dan efisiensi operasional
bulanan perusahaan khususnya pada periode Oktober 2025.

## Deliverables

- **PBIX**: ButterflyEffect_Dashboard.pbix
- **PDF**: ButterflyEffect_Dashboard.pdf

## Insight Utama (Oktober 2025)

Secara keseluruhan, kinerja perusahaan pada Oktober 2025 mengalami **penurunan signifikan**. _Total Net Sales_ menurun **64%** menjadi **$50.07K**, dipicu oleh berkurangnya jumlah pesanan sebesar **34.3%**. Meskipun kategori _Beverages_ tetap menjadi kontributor utama, tren bulanan mencapai titik terendah setelah puncak di September. Terdapat **kenaikan Average Discount sebesar 25.3%** menjadi **5.64**, yang mengindikasikan bahwa strategi diskon tinggi belum efektif mendongkrak volume penjualan bulan ini. Di sisi operasional, perusahaan tetap efisien dengan rata-rata waktu kirim **4.9 hari**. **Federal Shipping** menjadi mitra tercepat (**4.3 hari**), namun biaya logistik ke **Jerman ($396.23)** perlu dievaluasi untuk melindungi margin laba. </p>
